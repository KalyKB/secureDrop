# secureDrop
Secure web-based file storage application with authentication, encryption, and access control designed for cybersecurity coursework.

In order to be able to use the file, download both the frontend and backend folders and place them in the Desktop, for example. Then, via the command prompt, locate the folders (C: User/example/example/backend), then enter (Node server.js). After that, the backend should be running.

Now access the frontend Example--> (VS Open with live server)

After signing in, normal users could be changed into admins via the database.
